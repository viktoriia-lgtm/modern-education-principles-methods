Вступ до CSS3 та веб-дизайну

CSS3 – це сучасна версія мови каскадних таблиць стилів, яка забезпечує стилізацію веб-сторінок. Вона додає багато інновативних можливостей для створення привабливих та функціональних веб-інтерфейсів.

Типографіка та Google Fonts

Google Fonts надає безкоштовну бібліотеку висококвалітетних веб-шрифтів, які можна легко інтегрувати у веб-сайти. Це дозволяє дизайнерам та розробникам використовувати привабливі шрифти без збільшення розміру сторінки.

Особливості типографіки CSS3:

  • color – встановлює колір тексту
  • font-size – визначає розмір шрифту
  • line-height – контролює висоту рядка (вертикальний інтервал)
  • text-align – вирівнює текст (left, center, right, justify)
  • font-family – вказує сімейство шрифту
  • font-weight – контролює товщину шрифту (400, 700, 900)

Створення гнучких макетів з Flexbox

Flexbox – це сучасний метод для створення адаптивних макетів у CSS. Він забезпечує простий та ефективний спосіб розташування елементів у контейнері.

Основні властивості Flexbox:

  • display: flex – активує режим flex для контейнера
  • flex-direction – визначає напрямок (row, column)
  • justify-content – вирівнює елементи по головній осі
  • align-items – вирівнює елементи по поперечній осі
  • flex-wrap – дозволяє елементам переходити на новий рядок
  • gap – встановлює відстань між елементами

Адаптивний дизайн та медіа-запити

Медіа-запити дозволяють адаптувати стилі під різні розмірори екрану та пристрої. Це гарантує, що сайт виглядає чудово на комп'ютерах, планшетах та смартфонах.

Точки перелому (Breakpoints):

  • Desktop: більше ніж 768px – макет з основним контентом та бічною панеллю
  • Tablet: 481px–768px – перехідна версія з коригуванням розмірів
  • Mobile: менше ніж 480px – вертикальний макет, бічна панель під контентом

Приклад медіа-запиту:

@media (max-width: 768px) {
  .main-container {
    flex-direction: column;
  }
  
  .main-content,
  .aside {
    width: 100%;
  }
}
