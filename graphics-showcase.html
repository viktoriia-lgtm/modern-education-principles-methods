Графічні елементи для освітнього сайту

Проєктування та інтеграція логотипу, банера та кнопок

1. ЛОГОТИП

Логотип з текстом

Іконка логотипу

Порада: Експортуйте логотип у декількох розмірах (600x200, 300x100, 150x50 px) для різних контекстів використання.

2. БАНЕР

Параметри:
• Розмір: 1920x1080px (співвідношення 16:9)
• Адаптивне масштабування для мобільних пристроїв
• Фон: Лінійний градієнт від #2C3E50 до #3498DB
• Декоративні елементи: Кола та логічні символи
• Текст: "Математична Логіка" + підзаголовок

Розміри файлів:
• PNG: 200-300 KB
• JPG (оптимізований): 100-150 KB

Рекомендація: Використовуйте JPG для вебу, щоб зменшити час завантаження сторінки.

3. ГРАФІЧНІ КНОПКИ

Набір трьох кнопок з єдиним стилем для основної навігації сайту

Параметри кнопок:
• Розмір: 300x120px (адаптивні до мобільних 200x80px)
• Форма: Прямокутник з заокругленими кутами (border-radius: 15px)
• Ефект: Градієнт та тінь для об'єму
• Шрифт: Open Sans Bold, 28px, білий
• Розміри файлів: 15-25 KB (PNG), 5-10 KB (SVG)

4. ПОРІВНЯННЯ ВЕФОРМАТІВ: JPEG, PNG, GIF

Аналіз різниці між основними графічними веб-форматами та рекомендації щодо вибору для кожного елемента

Формат Компресія Прозорість Кількість кольорів Розмір файлу Найкраще для
JPEG Втратна Ні 16,7 млн 20-100 KB Фотографії, градієнти, сложні зображення
PNG Без втрат Так (8 або 32 біти) 16,7 млн 50-300 KB Логотипи, іконки, прозорість
GIF Без втрат Так (1 бік) 256 10-200 KB Анімовані зображення, прості іконки
SVG Векторна Так Необмежено 5-50 KB Логотипи, іконки, масштабовані графіки

Рекомендації для кожного елемента:

Логотип з текстом:
✓ PNG - Прозорий фон, без втрати якості, ідеально масштабується
✓ SVG - Найкраще рішення, векторний формат, безмежне масштабування
✗ JPEG - Немає прозорості, видні артефакти при компресії
✗ GIF - Обмежена палітра кольорів (256)
Іконка логотипу:
✓ PNG - Малий розмір, чітка видимість навіть при малих розмірах
✓ SVG - Оптимально для фавіконів, идеальна для всіх розмірів
✗ JPEG - Невідповідний для простих графічних елементів
Банер (1920x1080px):
✓ JPEG - Добра компресія, менший розмір файлу (100-150 KB)
✓ PNG - Краща якість, але більший розмір (200-300 KB)
✗ GIF - Неминуче буде дуже великим файлом через кількість кольорів
Графічні кнопки:
✓ PNG - Прозорість, чіткі лінії, добрий для вебу
✓ SVG - Найкраще рішення, безмежне масштабування, лічні взаємодіяння (hover)
✗ JPEG - Видні артефакти, немає прозорості

Загальні рекомендації з оптимізації:

  • Логотип: PNG (для вебу) + SVG (як основний формат для масштабування)
  • Банер: JPEG з якістю 85-90% (кращий баланс якості та розміру)
  • Кнопки: SVG (ідеальне масштабування) або PNG (для швидшого завантаження)
  • Оптимізація розміру: Використовуйте TinyPNG/TinyJPG або ImageOptim
  • Мобільна адаптивність: Завантажуйте менші версії для мобільних пристроїв
